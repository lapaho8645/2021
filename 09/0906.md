## 구조체 전송
소켓을 이용하여 구조체 데이터를 전송
#### 클라이언트
```c
struct Info{
  int age;
  char name[100];
};
struct Info user;
user.age = 10;
strcpy(user.name, "박");

send(sock,(char*)&user,sizeof(struct Info),0);
```
#### 서버
```c
struct Info{
  int age;
  char name[100];
};
struct Info user;
char buf[512];
int len;
len = recv(cli_sock, buf, 512, 0);
buf[len] = '\0';
user = (struct Info*)buf;
printf("name : %s\n", user->name);
```
